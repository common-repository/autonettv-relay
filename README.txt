Delivers AutoNetTV Media, Inc. blog posts to your wordpress site through the Relay API.

== Description ==

This plugin connects you to your AutoNetTV blog posts through the Relay API. If you wish to alter the content of your posts,
remember to do it through your online admin portal not through wordpress. If you alter the posts through WordPress, the content will be overwritten each time the plugin checks for updates.

== Installation ==

Required
1) Collect your API domain and key information from your online admin portal.
2) On the wordpress admin menu under AutoNetTV, enter in your domain and key, then press save changes.
3) Click Category Selections to select which categories you would like to import.
4) Choose your article sync schedule. If you only receive one article a week, please set that to weekly or daily.

== Disclaimer ==

This plugin removes jquery from the basic installation of WordPress and may effect your theme. By using this plugin this must be done to play the videos at the current time.
